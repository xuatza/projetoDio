História da Web

(Voltar)

A história da WWW, é uma parte essencial da história da Internet. A web é o sistema de informação que permite que documentos, imagens e outros recursos sejam vinculados e acessados através de hipertexto e hiperlinks. Teve seu surgimento graças aos primeiros computadores, como ENIAC,IBM e também a nomes importantes como Robert E. Kahn, Tim Berners-Lee e Bill Gates.

“A internet é muito mais que uma tecnologia. É um meio de comunicação, de interação e de organização social.”
Manuel Castells

O que são Clients

(Voltar)

Um "client" refere-se a um dispositivo, aplicativo ou programa que solicita e recebe serviços, recursos ou informações de um servidor.Podem ser navegadores como, Google Chrome,Mozila Firefox ou Microsoft Edge, podem ser também dispositivos como, smartphones, computadores ou tablets.

O que são Servers

(Voltar)

Um servidor é um dispositivo ou software dedicado a fornecer serviços, recursos ou informações para outros dispositivos chamados de clientes. Os servidores desempenham um papel crucial na infraestrutura da Internet e em redes de computadores em geral. Entre os principais tipos de servidor estão, servidor Web, de Email, de arquivos, de banco de dados...

Linguagens de Programação

(Voltar)

São uma forma de o ser humano interagir com a máquina e passar instruções para a mesma.Existem as linguagens Server-Side, que são executadas no servidor, C#, Java e PHP.Existem também as linguagens Client-Side que são executadas diretamente no dispositivo do usuário final, JavaScript,HTML,CSS.A escolha da linguagem depende do tipo de aplicativo que você deseja desenvolver, dos requisitos do projeto e das preferências pessoais. Cada linguagem tem suas próprias vantagens e desvantagens.

Estrutura Básica HTML

(Voltar)

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem fundamental para a criação de páginas da web. Ele define a estrutura e o conteúdo de uma página web por meio de elementos e tags HTML.

Tags

(Voltar)

São elementos usados para marcar e estruturar o conteúdo em uma página da web, são usadas para indicar ao navegador como o conteúdo deve ser renderizado. Existem básicamente 3 tags consideradas padrão no html.São elas:

  1. HTML: Define que o programa é uma página html
  2. Head: Contém informações sobre o documento como por exemplo o título, não é responsável pelo conteúdo visível da página.
  3. Body: É a responsável por conter todas as informações que serão visualizadas pelo usuário.

Atributos de Tags

(Voltar)

São propriedas que uma tag pode ter, existem atributos que são globais e existem os que são específicos de algumas tags. Os atributos são usados para controlar o comportamento, a aparência ou outros aspectos dos elementos na página da web. A maioria das tags HTML pode ter atributos, embora nem todas as tags aceitem os mesmos atributos.

Textos

(Voltar)

Os textos são parte essencial do conteúdo de uma página da web. Eles podem ser usados para apresentar informações, criar parágrafos, criar cabeçalhos, definir listas e muito mais.Para incluir textos em html podemos utilizar parágrafos com p, cabeçalhos com h1 até o h6, negrito com strong ...

Listas ordenadas e não ordenadas

(Voltar)

São consideradas os principais tipos de listas de html.Listas Não Ordenadas ul, é usada quando a ordem dos itens na lista não é importante, e os itens são geralmente marcados com marcadores (como pontos ou círculos). Listas Ordenadas ol, é usada quando a ordem dos itens na lista é importante, e os itens são numerados automaticamente em ordem crescente ou em ordem alfabética.

Links

(Voltar)

Em HTML, você pode criar links para outras páginas da web, recursos externos, arquivos locais e diferentes partes da mesma página usando a tag 'a' (âncora). Existem links para página da web externa, links internos em uma página, links para arquivos ou links de email, além disso é comum adicionar outros atributos, como target para controlar como o link é aberto (em uma nova janela, na mesma janela, etc.), e rel para especificar o relacionamento do link com a página atual
